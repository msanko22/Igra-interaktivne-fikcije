# Igra-interaktivne-fikcije Smiljan story
Igra interaktivne fikcije - Smiljan Story - projekt iz predmeta Deklarativno prograiranje

#Pokretanje igre
1. Preuzmite cijeli repozitorij na svoje rečunalo (klik na gumb **Code** zatim **Download**
2. Raspakirajte datoteke ujedan folder
3. Otvorite datoteku "index.html" pomoću internetskog preglednika (npr. Edge)
4. Nakon što se stranica učita, odaberite opciju **Play in browser** kako biste započeli igrati igru.
